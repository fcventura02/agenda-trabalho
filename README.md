## Projeto

Projeto realizado durante o workshop da codar.me, esse projeto foi trabalhado utilizando nextjs e firebase, sendo estudado tratamento de data, requests, rotas privadas e publicas.

## O que foi utilizado

- firebase
- firebase-admin
- date-fns
- formik
- yup
- chakra-ui
- axios
